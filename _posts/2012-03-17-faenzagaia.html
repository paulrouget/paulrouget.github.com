I want a Faenza-based Gaia theme.

If you don't know Gaia…

Mozilla is building an Mobile OS, B2G. On top of this OS, we are also building a User Interface, named Gaia (see some mockups).

If you don't know Faenza…

Faenza is a famous set of icons for Linux. His author, Tiheum (Matthieu James), has also built a theme for Gnome (Faience), and made a lot of other gorgeous things.

I am a huge fan of Tiheum's work. And I believe that a phone UI based on his work would be a hit. I would love to work on that, but I don't have time. So I am sharing the idea. It is quite easy to hack Gaia to build a prototype. It is 100% made of HTML, CSS and JavaScript. So if you feel like working on that, please let me know, I can help you to start :)
