Mozilla and Github in 2013

Github shared some interesting data about repositories activity in 2013: http://octoverse.github.com/. And apparently the project with the most merged pull requests is a Mozilla project: Gaia (Firefox OS frontend), and the 2nd repository with the most issues closed is also a Mozilla project: Rust (a programming language).
