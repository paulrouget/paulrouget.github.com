CSS3 Animations in Firefox Aurora

David Baron implemented CSS3 Animations in Firefox. Yeah!. You can test CSS3 Animations right now with Firefox Aurora (don't know what Firefox Aurora is? Check this out).

If the implementation is good enough (please report bugs!!!), Animations will make it for Firefox 5.
