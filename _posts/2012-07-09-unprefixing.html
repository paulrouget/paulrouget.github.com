CSS3 Animations, Transitions, Transforms and Gradients to be unprefixed in Firefox 16

In Firefox 16, we are unprefixing:

  • CSS3 Animations bug 762302
  • CSS3 Transitions bug 762303
  • CSS3 Transforms bug 745523
  • CSS3 Gradients bug 752187
  • IndexedDB bug 726378

calc() might be unprefixed as well (bug 771678).
