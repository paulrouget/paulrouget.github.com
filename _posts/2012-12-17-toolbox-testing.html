DevTools Toolbox ready for testing

For the past 2 months, we have been working on a new user interface for the Firefox Developer tools. As explained previously , we are building a unified interface for our tools (the "toolbox").

You can now test this new UI in Firefox Nightly. Because this feature required a lot of code refactoring, we are expecting regressions (see known bugs). Please help us to test the toolbox. If you run into issues, please file a bug (Developer Tools components), or leave a comment here.
