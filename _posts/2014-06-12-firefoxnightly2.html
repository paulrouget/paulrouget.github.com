@FirefoxNightly twitter account - how you can help

The goal of the twitter FirefoxNightly account is to regularly tweet about interesting bug fixes (user facing and web platform features, and performance improvements), interesting blog posts or interesting new articles on MDN.

It's not as simple as just "tweeting". Bugs need to be found, tweets need to be scheduled, tweets need to be RT by mozillians in their respective timezones, bugs need to be validated (is the fixed actually landed in Firefox Nightly, and is it behind a pref), tweets need to be redacted (the bug summary is not always enough), maybe a small demo or a screenshot would help, tweets need to be synced with our tumblr blog, ...

This is where I need help:

  • Find "things" to tweet about and add them to etherpad.mozilla.org/firefoxnightly. This is a thing I'm good at and I'll keep doing it;
  • Figuring out how what tool we should use to schedule the tweets and sync them with tumblr (buffer, tweetdeck, IFTTT, ...);
  • Make sure tweets get retweeted during the most important timezone;
  • Validate the bug and figure out if a pref needs to be switched or not;
  • If needed, build a short demo (I use jsbin.com. If it's a web platform feature, I can do it;
  • If needed, take a screenshot (I use imgur.com);
  • Redact and feed the tweet to twitter.com or to any tool we might use to schedule the tweets

If you're interested to help, please let me know.
