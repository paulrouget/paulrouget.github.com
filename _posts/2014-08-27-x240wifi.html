Intel 7260 WiFi issues (thinkpad 240)

Getting disconnected often? Slow connection? Check this out:

To check what is your wireless chipset: lspci | grep Wireless. If it's an Intel 7260. Check what firware is used: dmesg | grep "iwlwifi .* loaded firmware version" (Don't trust modeinfo, it's broken). If you don't see any output, reboot. If the firmare version is 22.24.8.0, you have firmware version 8. Which is terribly buggy. You want to use version 7. To do so, back-up /lib/firmware/iwlwifi-7260-8.ucode, and then delete it. Reboot.
